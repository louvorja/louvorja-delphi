For Delphi 5-XE7:

 - open bsfd*.dpk file
 - install package

For C++Builder 5, 6:

 - open bsfcb*.bpk file
 - install package

For C++Builder 2006-XE7:

 - open *.cbproj file
 - install package

See bsfsb.zip with SkinBuilder sources.


Note:

  if you use BusinessSkinForm VCL in Delphi 2007-XE7 or C++Builder 2007-XE7
   you must open project sources in designing time and
   set Application.MainFormOnTaskBar to False!
